# Gesture Images

This directory should contain PNG files for Sicilian/Italian gestures.

## Required Files

The following gesture image files are referenced by the application:

- `mano_a_borsa.png` - "What do you want?" gesture (fingertips together)
- `corna.png` - Horn gesture (index + pinky extended)
- `ma_che_vuoi.png` - "What are you saying?" gesture
- `vieni_qua.png` - "Come here" gesture (downward palm wave)
- `piano_piano.png` - "Slowly/carefully" gesture (flat hand moving down)
- `basta_stop.png` - "Stop/enough" gesture (raised palm)
- `perfetto_ok.png` - "Perfect" gesture (fingertip kiss)
- `prometto_giuro.png` - "I swear" gesture (hand on heart or raised)
- `vattene_via.png` - "Go away" gesture (back of hand flick)
- `silenzio.png` - "Quiet/silence" gesture (finger to lips)

## Format

- **Type**: PNG (transparent background preferred)
- **Size**: 400x400px or similar square aspect ratio
- **Style**: Clear, illustrative photos or drawings showing hand position
